# koydo-tesda-nc

TESDA NC exam preparation — powered by Koydo.

## Quick start

```bash
npm install
npm run dev
```

## Stack

- **Framework:** Next.js 16.2.0 + React 19
- **Styling:** Tailwind CSS 4
- **Backend:** Supabase (shared)
- **Deploy:** Vercel

## Exam Metadata

| Field | Value |
|-------|-------|
| Exam ID | EXAM086 |
| Name | TESDA NC |
| Country | Philippines |
| Category | Vocational certification |
| Tier | Tier 6 |
| Wave | Wave 8 |

This app is a study aid based on publicly available syllabus and exam patterns. Not affiliated with any official examination authority.
